🚀 扫雷游戏 PWA 部署向导

📦 部署选项

选项 1: Netlify 部署 (推荐)

✅ 最简单的部署方式

方法 A: 拖拽部署

1. 访问 Netlify Drop

2. 将整个项目文件夹拖拽到页面上

3. 自动获得部署网址

方法 B: VS Code 命令

1. 在 VS Code 中按 Ctrl+Shift+P

2. 输入 "Tasks: Run Task"

3. 选择 "部署到 Netlify"

或在终端运行：

npm run deploy:netlify

选项 2: Vercel 部署

✅ 快速且性能优秀

在终端运行：

npm run deploy:vercel

或在 VS Code 任务中选择 "部署到 Vercel"

选项 3: GitHub Pages

✅ 免费且与 GitHub 集成

1. 推送代码到 GitHub

2. 在仓库设置中启用 GitHub Pages

3. 选择 main 分支作为源

🔧 VS Code 快速操作

使用命令面板

按 Ctrl+Shift+P 打开命令面板，然后选择：

  • Tasks: Run Task → 启动开发服务器
  • Tasks: Run Task → 部署到 Netlify
  • Tasks: Run Task → 部署到 Vercel

使用终端命令

# 启动开发服务器
npm run dev

# 部署到 Netlify
npm run deploy:netlify

# 部署到 Vercel
npm run deploy:vercel

📱 PWA 安装测试

测试步骤

  1. 部署完成后，获得网址（例如：https://amazing-game-123.netlify.app）
  2. 在手机浏览器中访问该网址
  3. 查看是否出现"添加到主屏幕"提示
  4. 安装后测试离线功能

PWA 检查清单

  • ✅ HTTPS 连接
  • ✅ manifest.json 文件
  • ✅ Service Worker
  • ✅ 图标文件
  • ✅ 响应式设计

📞 获取帮助

如果遇到问题，可以：

  • 检查浏览器开发者工具的 Console 面板
  • 查看 Application → Manifest 检查 PWA 配置
  • 查看 Application → Service Workers 检查缓存
