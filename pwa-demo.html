🎯 第一步：快速部署到 Netlify

方法 A：拖拽部署（推荐初学者）

1. 打开 Netlify Drop

2. 将您的整个扫雷游戏文件夹拖拽到页面上

3. 等待上传完成，获得部署 URL

示例 URL: https://amazing-minesweeper-123456.netlify.app

方法 B：命令行部署（推荐开发者）

# 安装 Netlify CLI
npm install -g netlify-cli

# 登录 Netlify 账户
netlify login

# 部署项目
netlify deploy --prod --dir .

方法 C：VS Code 一键部署

1. 按 Ctrl+Shift+P 打开命令面板

2. 输入 "Tasks: Run Task"

3. 选择 "部署到 Netlify"

或者在终端运行：

npm run deploy:netlify

🔍 第二步：使用 PWABuilder 分析

打开 PWABuilder

访问 PWABuilder.com

输入您的网站 URL

在首页的输入框中输入您刚才部署得到的 URL，例如：

https://your-minesweeper.netlify.app

然后点击 "Start" 按钮

等待分析完成

PWABuilder 会检查以下项目：

  • ✅ Web App Manifest (manifest.json)
  • ✅ Service Worker (sw.js)
  • ✅ HTTPS 连接
  • ✅ 图标文件
  • ✅ 响应式设计

📦 第三步：生成安装包

🤖 Android

生成文件：

  • APK - 直接安装
  • AAB - Google Play Store

用途： Android 手机和平板

优势： 无需审核，立即可用

🍎 iOS

生成文件：

  • Xcode 项目文件

用途： iPhone 和 iPad

要求： macOS + Xcode + 开发者账户

🪟 Windows

生成文件：

  • MSIX 安装包

用途： Windows 10/11 桌面

优势： 完整桌面应用体验

🍎 macOS

生成文件：

  • PKG 安装包
  • DMG 磁盘映像

用途： Mac 桌面应用

优势： 原生 macOS 体验

生成步骤

1. 分析完成后，点击 "Package For Stores"

2. 选择要生成的平台（可以选择多个）

3. 配置应用信息（名称、图标、描述等）

4. 点击 "Generate" 开始生成

5. 等待处理完成（通常 5-10 分钟）

6. 下载生成的安装包文件

🧪 第四步：测试安装包

Android APK 测试

1. 下载 .apk 文件到 Android 设备

2. 在设置中开启"允许未知来源安装"

3. 点击 APK 文件进行安装

4. 安装完成后，在桌面找到应用图标

5. 测试游戏功能和离线使用

Windows MSIX 测试

1. 双击 .msix 文件

2. 跟随安装向导

3. 从开始菜单启动应用

4. 测试桌面应用体验
